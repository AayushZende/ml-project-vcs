# ml-project-vcs
machine learning project using github for version control.
## Projet Goal
This project demonstrates the use of GitHub as a Version Control System(VCS)
for a MachineLearning Workflow.The Goal is to manage Datasets, code, and collaboration efficiently.

##Dataset
The dataset used is a CSV file containing sample data for machine learning
experiments such as classification or regression.

## Tools Used
-Python
-Git and GitHub
-CSV Dataset
